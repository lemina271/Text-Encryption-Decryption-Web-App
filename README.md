Text Encryption & Decryption Web App

A Flask-based cybersecurity web application that allows users to encrypt and decrypt text using multiple cryptographic algorithms through a clean, dark-themed, and user-friendly interface.

This project is built for learning **web security**, **cryptography basics**, and **secure application development**.

---

Features

- Encrypt and decrypt text in real-time
- Multiple encryption algorithms
- Dark professional UI
- Simple and user-friendly design
- Client-side & server-side validation
- Built using Flask, HTML, CSS, and JavaScript

---

Encryption Algorithms Used

1. **Caesar Cipher**
2. **Base64 Encoding**
3. **AES (Advanced Encryption Standard)**
4. **SHA-256 Hashing**
5. **Reverse Cipher (Basic transformation)**

---

Technologies Used

- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, JavaScript
- **Styling:** Custom Dark Theme CSS
- **Version Control:** Git & GitHub
